001 Git เรียนรู้การใช้ git

git init สร้าง.git folder

git add . เพิ่มfileเป็นสถานะ stage .คือการเอาทั้งหมดในdirectory

git config --global user.email "you@example.com" set account's default identity
git config --global user.name "Your Name"

git commit -m "first commit" commit file พร้อม messsage ใน""

git remote add origin https://github.com/ZalotBabyZ/PreCodecamp-7.git เชื่อมต่อกับ repository

git status check ตน.

git push -u origin master push พร้อมระบุตำแหน่งที่จะuploadขึ้นไปบนrepository
git push รู้ตำแหน่งที่จะpushไปแล้ว ไม่ต้องระบุใหม่

git add . -> git commit -m "message" ถ้าไม่มีการเปลี่ยนแปลงใดจะcommitไม่ได้ -> git push
